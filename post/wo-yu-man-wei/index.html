我与漫威

Posted on 2019-05-31，10 min read

结识 Marvel

高二入的坑，现在快大四了。

当时跟着朋友去电影院看了《复联2》，这是唯一一场我忘记带眼镜去看的电影（之前有听说过钢铁侠、黑寡妇等角色），字幕啥的也没看清楚，角色也是模糊的，只知道后半部分打打杀杀。

之后也不知道为什么又在电脑又看了一遍，那时，惊艳于寡姐的性感、铁人的科技、浩克的无敌，以及美队的信念。于是，那年暑假，补完了所有漫威电影，包括不属于MCU的X战警、神奇四侠系列电影，还有《神盾局》和《卡特》。接着，高三5月晚自习去看了《美队3》；高考前几天去看了《天启》。因为不看的话，高考时肯定要胡思乱想。

或许高中的我，喜欢漫威的原因很简单，只是热衷于英雄主义，盲从着好莱坞带来的无限视效。

Marvel 情结

上大学后，我关注上了电影本身，远不限于漫威。

我欣赏着《末代皇帝》中的光影与色彩，陶醉于那段历史的魅力，我思考着《血观音》的人性与黑暗，我热爱着《美丽人生》的温情与感动，我沉浸在皮克斯动画电影带来的各种奇妙设定......所以对于电影院上映的电影，我一直有自己的观影风向，我选择看的电影，一般有以下几种：
1.有正确口碑的认证，确实值得一看，例如《我不是药神》、《海王》；
2.曾经看过的好电影的续集，例如最近上映的《玩具总动员4》；
3.对该片的团队或导演十分信任，比如皮克斯工作室、卡梅隆导演；
4.有漫威角色的电影，例如MCU、XCU、索尼家的《毒液》。

不难理解，第4点是我的Marvel情结。

所有的商业电影里，除了个别确实值得看的电影外，我确实是只吃漫威这一套，就算漫威出了部再差的片，我也会去电影院支持，但看完后，我依然会辩证看待这部电影。

《黑豹》剧情单薄但多个角度塑造了非洲未来科技美学；福斯拍的《X战警：黑凤凰》打戏流畅、特效很足，但多处情节转换僵硬，结局走向也不明不白；而索尼拍的《毒液》是真的低于我个人的及格线了。

我知道漫威是商业电影，它也从来没想在电影史上留下什么深刻的笔划，它以漫改电影的形式向世界传播着他们上世纪创造出来的超级英雄们，它确实也专注于自身，漫威在我的观影库中充当的是“电影美剧”，这部剧每集基本稳定发挥，运作成熟，每年来个2～3集，有时确有失望，有时也差强人意，有时又突然有了议题。

作为MCU粉丝以及电影爱好者，这样的身份给我提供了一个格外清晰的视角。

树大难免招风，世界在进步，少部分人的意识越来越自主，他们知道漫威电影缺失深度且知道缺失在哪，偶尔会给出客观的批评或中肯的建议。

从电影的角度来说，超英题材在影史上确实有佳作，虽然《复联4》目前已经成为了世界影史票房第一的作品，超英的票房天花板可能还未出现，未来一定可以触及且再度突破，但超英的天花板估计已经是止于《蝙蝠侠：黑暗骑士》了，这部作品对于正义存在的探讨以及超英人格的全面开启实在是令人叹服，着实难以超越。

大多数人选择跟风。

踩它的人说电影有特效没剧情以及超英主题审美疲劳，却没想到复联3的英雄阵亡和复联4的时间穿越串起了多少人的期待，这个电影宇宙就是它特有的故事。

捧它的人亦是信仰难挡，漫威票房神话是真，流水线制作也是真，只是还没出现极具败相的电影，没必要每部都五星好评似的疯狂吹捧，这只会有碍于它在未来的进步，还有英雄退场是发展必然，没必要拘泥于角色上的死亡，毕竟这个宇宙还有很多故事没讲。

《复联4》的首映与二刷

我没想到，有一天，我会熬夜去看首映，我会考虑在影院二刷一部电影。

是的，《复联4》是我人生中第一部首映，以及第一部影院二刷的电影。我从未想过我会疯狂到熬夜去看一部长达3小时的电影，但真的难掩激动。从定档到预售，再到观影前夕，二十几天的时光，是难熬的，也是狂奔的。

首映有着绝对的魅力。

24号零点场带给我的感受主要来自于观影者。

观影前夕，进电影院后看到陆陆续续取票的观众，看到那撞衫率超高的衣服标志，看到不断被用来提神的湿巾，看到钢铁侠雕像不间断的合影留念。

从电影的开头到结尾，一群人的欢乐、惊喜、悲伤、失望交织在同一时间点，同时动容，全场欢呼，同仇敌忾，与有荣焉，一群陌生人悲欢相通，这是件奇妙而又幸福的事。

二度观影，更是沉浸。

从严格的观影意义上来说，《复联4》不是我第一部“二刷”的电影。

我始终认为，时间应该留给新鲜的东西，但对我有绝对魅力的电影，我一定会再去过目几遍。《机器人总动员》是我小学六年级看的电影，它对我有很大的影响，时至今日我仍会抽出时间来看看它的许多片段，我一直关注着动画电影，对动画电影的信任大部分源自它；《美丽人生》我看了4遍，它着实让我动心，并打算往后每年都看一遍；《龙猫》我在视频网站上看了一次，去年又去电影院支持了一遍...

所以，准确来说，《复联4》是我第一部在电影院二刷的电影。

二刷的《复联4》是在杜比影院看的，对比第一场的普通厅，观影体验肯定是提升不少，这影院确实不同于普通影院，大屏清晰，声感震撼，座椅舒适。除了视效上的提升之外，二刷的我很明显地倾向于关注牺牲者了，我更悲伤于寡姐在飞船上的笑了，我更在意小辣椒和摩小根眼神里对托尼的爱意了，我更心疼队长的每一次摔落和爬起，虽然他是人类身体的极限.....

至于为什么二刷？原因就这几个。

  1. MCU粉丝因素。
  2. 上文提到的，《复联4》对我有的绝对魅力。
  3. 陪还没看的朋友去电影院看。
  4. 理清一些第一遍没看懂的情节，着重于观察《复联4》的时间穿越观。
  5. 再看一遍对我有独特情怀的“彩蛋们”。
  6. 找首歌，浩克和火箭前往新阿斯加德的BGM。已找到，The Kinks的《Supersonic Rocket Ship》。
  7. 再度告别三个凡人：铁人、美队、寡姐。

《复仇者联盟4》观后感

对于路人来说，首映和二刷都是不太现实的。
首先是《复联4》的时长和票价确实劝退，其次是上映以来路人最多的反映就是文戏太长且大战略短。

对比《复联3》的情节紧凑和叙述完整，《复联4》埋梗众多，再加上独特且复杂的时间穿越，又杂揉了多段有关无限宝石的历史，把MCU所有电影都看过的观众都不一定能反应到某个点的精彩，《复联4》大概是MCU至今最粉丝向的电影了，此次确实是非常不照顾路人。

对于我来说，《复联4》不只是22部电影的集结，不止于我的Marvel情结。

《复联4》集MCU所有电影之大成。

开篇着重于《复联3》的响指危机和复联幸存成员的现状，动用量子力学进行时间穿越，有条不紊地完成，并铺陈出漫威后续布局的多元宇宙；后半部分的大战空前盛大，虽然与《复联3》有巨大反差的战位分布，但也短而充足地展现了每位超英在现阶段各个任务的能力，主要集中在“三巨头”，新英雄的亮相反倒不多，反派主要集中在展现无宝石灭霸的超强战斗力，由此也削弱了黑曜四将的存在感，无限宝石的能力在《复联4》中主要集中在响指的使用以及一念之间的存亡感，不过收集宝石的过程中，有得有失，也揭示了未来多元宇宙的到来。

作为粉丝，有笑点泪点，有众多彩蛋，有感动温度，有尚好结局；作为观众，有背景支撑，有剧情逻辑，有震撼观感，有观影议题。

这部电影给予我的绝对魅力是由点及面的。

#兴奋点

  1. “老朋友”们的现身，卡特、霍华德、贾维斯、古一、洛基。
  2. 美队举起雷神锤
  3. 奇异博士把所有英雄传送过来的时候
  4. 复仇者集结，大战灭霸军团
  5. 钢铁侠和小辣椒（救援战甲）并肩作战
  6. 猩红女巫单虐灭霸
  7. MCU所有女英雄集结（独缺寡姐，但这一幕我永远会想起她）

#惊讶点

  1. 惊奇队长新发型（致敬漫画）
  2. 雷神的胖造型
  3. 美国队长：九头蛇万岁
  4. 洛基带走空间宝石
  5. 雷神锤再现

#感动点

  1. 雷神妈妈：“你在未来一定吃了很多苦吧”
  2. 钢铁侠、摩小根：“我爱你 3000 遍”
  3. 鹰眼与黑寡妇在宇宙飞船的相视一笑
  4. 黑寡妇跳悬崖换取灵魂宝石
  5. 火箭浣熊在炮雨齐飞的时候护住格鲁特
  6. 钢铁侠葬礼

谈谈未来

说来惭愧，我也不能算得上是漫威的超级疯狂粉丝。

我不会关于漫威的每部电影都去二刷多刷的支持，我不会去购入与漫威相关但对我无价值的物品，我也会去看对手家的好电影，我也会在豆瓣上对无感的漫威电影打上中差评......

但我愿意为其花上一篇文章的时间，讲述那个热血狂燃的鲜红色标志。

就算铁甲破损，就算乏味剧本，就算脱离安稳，但那是众多个钢铁意志般的灵魂，是火红的永恒，是美漫文化的永生。

十年漫改，并未终章。

漫威宇宙开启十年，我错过了整整的前5年，后5年路过了我的高中和大学，《复联4》兴许就是那把开启多元宇宙的钥匙，漫威还有好多故事等待展开，它的未来，或许有一天我会疲劳，但我绝不会厌倦，更不会缺席。
